# Simple Web Clock

This is a simple web-based clock application that displays the current time and updates every second.

## Project Structure

- `index.html`: The main HTML file that contains the structure of the web page.
- `styles.css`: The CSS file for styling the clock.
- `script.js`: The JavaScript file that handles the clock logic.

## Setup & Installation Guide

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd simple-web-clock
   ```

2. **Open the application**:
   - Simply open `index.html` in your web browser.

No additional dependencies or backend setup is required for this project.
